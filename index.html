SpotTrack Bot

ربات تلگرام هوشمند برای دانلود مستقیم موزیک‌های مورد علاقه‌تان از اسپاتیفای با بالاترین کیفیت.

مشاهده در گیت‌هاب افزودن به تلگرام

قابلیت‌های شگفت‌انگیز

اطلاعات کامل آهنگ

دریافت نام، خواننده، آلبوم و کاور با کیفیت بالا فقط با ارسال لینک اسپاتیفای.

دانلود با کیفیت MP3

تبدیل و دانلود آهنگ‌ها به فرمت MP3 با کیفیت ۱۹۲kbps برای بهترین تجربه شنیداری.

سیستم پروکسی چرخشی

استفاده از پروکسی‌های چرخشی برای افزایش پایداری و دور زدن محدودیت‌های اسپاتیفای.

آرشیو هوشمند

آهنگ‌های دانلود شده در کانال تلگرام آرشیو می‌شوند تا در درخواست‌های بعدی فوراً ارسال شوند.

پشتیبانی از کاربران VIP

مدیریت محدودیت دانلود روزانه برای کاربران عادی و ارائه دسترسی نامحدود به کاربران VIP.

رابط کاربری تعاملی

تجربه کاربری لذت‌بخش با کیبوردهای تعاملی، پشتیبانی چندزبانه و انیمیشن پیشرفت دانلود.

معماری و جریان کار

کاربر
ربات تلگرام (Aiogram)
Spotify API (با پروکسی)
Downloader (yt-dlp)
MongoDB

نصب و راه‌اندازی

۱. کلون کردن پروژه

Bash
git clone https://github.com/MohammadHNdev/Spottrack-bot.git
cd Spottrack-bot

۲. نصب پیش‌نیازها

Bash
# Create and activate virtual environment
python3 -m venv venv
source venv/bin/activate

# Install Python packages
pip install -r requirements.txt

# Install ffmpeg (on Debian/Ubuntu)
sudo apt update && sudo apt install ffmpeg

۳. تنظیمات و پیکربندی

یک فایل config.py بسازید و مقادیر زیر را با اطلاعات خودتان پر کنید.

Python
TELEGRAM_BOT_TOKEN = "YOUR_TELEGRAM_BOT_TOKEN"
SPOTIPY_CLIENT_ID = "YOUR_SPOTIPY_CLIENT_ID"
SPOTIPY_CLIENT_SECRET = "YOUR_SPOTIPY_CLIENT_SECRET"
ARCHIVE_CHANNEL_ID = -100123456789  # آیدی عددی کانال آرشیو
PROXIES = [
    "http://user:pass@ip:port",
    # ...
]
MONGO_URI = "mongodb://localhost:27017"
# ... سایر تنظیمات
